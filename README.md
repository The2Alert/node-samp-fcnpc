## Installation
<p>Install package:</p>

```
npm i @sa-mp/fcnpc
```

<p>Add to list of plugins in config file <code>samp-conf.[json|json5|yml|yaml]</code>:</p>

```json5
{
    // <...>
    plugins: [
        // <...>
        "@sa-mp/fcnpc"
    ]
}
```
